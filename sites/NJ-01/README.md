# Draftly — Site Setup

## Placeholders to fill in (`index.html`)

| Placeholder | What to put |
|---|---|
| `YOUR-DISTRICT` | Your congressional district, e.g. `CA-12` |
| `YOUR TAGLINE HERE` | One-sentence description of the app |
| `ABOUT THE APP` | App description (2–3 paragraphs) |
| `VIDEO_ID` | YouTube video ID (the part after `?v=`) |
| `TECHNOLOGY 1–5` | Your tech stack — add or remove `<li>` tags as needed |
| `CAPTION 1–4` | Screenshot captions — add or remove `<figure>` blocks as needed |
| `ABOUT THE TEAM` | Team bio: school, grade, etc. |
| `TEAM MEMBER NAME` / `ROLE / GRADE` | Each team member's name and role |
| `#` on link cards | Real URLs for App Store, Play Store, GitHub, Website |

## Assets to drop in

| File | Where |
|---|---|
| Logo | `assets/logo.png` |
| Screenshots | `assets/screenshots/screen1.png`, `screen2.png`, … |
| Team photos | `assets/team/member1.png`, `member2.png`, … |
